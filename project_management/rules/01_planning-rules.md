# Planning Rules (Hard)

Phases:
- Phase 0: engineering gold standard + scaffolding
- Phase 1: prototype MVP (user feedback required)
- Phase 2: production readiness
- Phase 3: enterprise roadmap and scale

For each phase, the AI must produce:
- Phase objectives
- Phase gate checklist with required evidence
- Epics and stories scoped to the phase

Phase gate rule:
- The system may not proceed to the next phase unless all gate items are complete OR explicitly waived by a human with recorded rationale in an ADR.
